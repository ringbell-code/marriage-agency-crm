# データインポート手順

## 📋 SQLでデータを一括投入する方法

### 1. Supabaseダッシュボードを開く
[https://supabase.com](https://supabase.com) にアクセスして、プロジェクトを選択

### 2. SQL Editorを開く
左サイドバーから「SQL Editor」をクリック

### 3. SQLファイルを実行

#### オプション1: 新規クエリで実行
1. 「New Query」をクリック
2. `supabase/import_data.sql` の内容をコピペ
3. 「Run」ボタンをクリック

#### オプション2: ファイルからインポート
1. `supabase/import_data.sql` ファイルの内容をコピー
2. SQL Editorに貼り付け
3. 実行

### 4. 確認
```sql
SELECT COUNT(*) FROM members;
```

で件数を確認

---

## 🔄 データの更新

既存データと重複する `member_id` がある場合、自動的に更新されます。

### 全データを削除して再投入する場合：

```sql
TRUNCATE TABLE members CASCADE;
```

を `import_data.sql` の先頭行のコメントアウトを外して実行

---

## 📝 注意事項

- **RLSを無効化している場合のみ**、このSQLが正常に動作します
- データ投入後、Next.jsアプリをリロードすると最新データが表示されます
- 開発環境のみで使用してください（本番環境では適切な権限設定が必要）
